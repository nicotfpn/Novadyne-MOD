# NovaDyne

Aerospace and combat vehicles mod for Minecraft NeoForge 26.1.2.

## Content

- **Exosuit Tier 1** — GeckoLib-animated chestplate armor with custom 3D model
- **Plasma Cannon** *(planned)* — Energy weapon

## Development

Built with NeoForge MDK + GeckoLib 5.5.1. Java 25 required.

```bash
./gradlew build
```
